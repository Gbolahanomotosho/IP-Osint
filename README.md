<h1 align="center">IP-Osint</h1>
  

 IP-Osint is a simple python script that fetch out some information about an ip address 
 
 Which includes geographical location üìç like Continent, Country, Region, City, Latitude, Longitude, Capital, Borders, Internet Service Provider (ISP) 
 
 ASN (Autonomous System Number), Country code, Region code, Postal code, Calling code, Timezone, Country flag, Google Maps üåç Link and so on

 It supports both IPv4 and IPv6 IP Address type...................
